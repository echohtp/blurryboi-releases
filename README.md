## What is BlurryBoi?

You're on a Zoom call. Slack is pinging. Your notes app has passwords visible. Your browser has 47 tabs of questionable productivity.

**BlurryBoi blurs all of it.** Only your focused window stays clear. One hotkey. Instant privacy.

## Install

1. Download the latest `.zip` from [**Releases**](https://github.com/echohtp/blurryboi-releases/releases/latest)
2. Unzip and drag `BlurryBoi.app` to Applications
3. Launch — grant Accessibility permission when prompted
4. Press **Ctrl + Option + B** to toggle blur

> Requires **macOS 13 (Ventura)** or later. No dependencies.

## Features

| Feature | Description |
|---|---|
| **Global hotkey** | Configurable toggle — default `Ctrl+Option+B` |
| **Peek mode** | Hold a key to temporarily reveal everything |
| **Click-through** | Click the blur to focus the window underneath |
| **Multi-monitor** | Blur overlay on every connected screen |
| **Focus timer** | Built-in Pomodoro with work/break cycles |
| **Schedule** | Auto-enable blur during your work hours |
| **Customizable** | Tint color, opacity, blur intensity, and material |
