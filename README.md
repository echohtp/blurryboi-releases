## The problem

You're screen-sharing on Zoom. Behind your slides: Slack DMs, personal email, a notes app with passwords, and 47 browser tabs of questionable productivity.

**BlurryBoi makes it disappear.** One hotkey and everything blurs out except the window you're working in. Click through the blur to switch focus. It's that simple.

## Quick start

```bash
# 1. Download from releases (or grab the .zip below)
# 2. Unzip → drag BlurryBoi.app to /Applications
# 3. Launch → grant Accessibility permission
# 4. Done.
```

<kbd>Ctrl</kbd> + <kbd>Option</kbd> + <kbd>B</kbd> — toggle blur on/off

That's it. No config files. No terminal. No dependencies.

> **Requires macOS 13 (Ventura) or later.**
